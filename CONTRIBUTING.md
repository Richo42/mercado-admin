# Cómo contribuir a este proyecto

1. Haz Fork de este repositorio
2. Clona tu fork: `git clone https://github.com/TU_USUARIO/mercado-admin.git`
3. Crea una rama para tu cambio: `git checkout -b feature/mi-cambio`
4. Realiza tus modificaciones y haz commit
5. Sube los cambios a tu fork: `git push origin feature/mi-cambio`
6. Abre un Pull Request desde GitHub explicando qué cambiaste

¡Gracias por contribuir! 🙌
